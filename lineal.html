Estructuras de Control

Las estructuras de control en programación son herramientas que hacen posible que los algoritmos sigan un flujo lógico de acontecimientos y que gestionen el comportamiento de un programa.

Case

Esta estructura se utiliza para seleccionar una de varias opciones posibles basadas en el valor de una variable. Es útil para simplificar múltiples condiciones

If anidado

Esto permite evaluar múltiples condiciones de forma jerárquica.

If

Esta estructura permite ejecutar un bloque de código solo si una condición específica es verdadera. Si la condición es falsa, el bloque se omite.

If/Else

incluye una segunda opción. Si la condición es verdadera, se ejecuta un bloque de código; si es falsa, se ejecuta un bloque alternativo.

do While

garantiza que el bloque de código se ejecute al menos una vez, ya que la condición se evalúa después de la ejecución del bloque.

While

Un bucle que ejecuta un bloque de código repetidamente mientras una condición específica sea verdadera. La condición se evalúa antes de cada iteración.

lineal

Este es el flujo básico de un programa donde las instrucciones se ejecutan secuencialmente, de arriba hacia abajo.
